## Project Description
This project is developed as part of the Web Technology (WT) subject.
It demonstrates the use of core web technologies by building a
static multi-page website with basic interactivity.

## Pages Included
1. **Biodata Page** – Displays personal and academic information  
2. **Gallery Page** – Shows images in a gallery format  
3. **Simple Calculator** – Performs basic arithmetic operations using JavaScript  

## Technologies Used

- CSS3
- JavaScript
